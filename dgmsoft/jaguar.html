Jaguar Virtual Video-Game Hardware System

Este é um projeto que pretendo retomar. Trata-se de um sistema de video-game virtual. A idéia é a seguinte:
Criar um emulador para um video-game virtual, como se fosse "real". Para tanto, idealizei seu processador, sua unidade de vídeo, de som, entrada e etc.
No entanto, meus limitados conhecimentos na área de hardware podem ter gerado muitas imperfeições no projeto :P
Porém ele ainda não está concluido e existem alguns fatores que estão em processo de alteração.
Caso você tenha interesse em me ajudar, dar sugestões, críticas, ou ainda escrever um jogo para o Jaguar, entre em contato comigo pela seção Contato. Vale lembrar que este projeto não tem nenhuma relação com o console Atari Jaguar :)

Para maiores informações sobre o projeto, por favor visite Jaguar Virtual Video-Game Hardware.
